# AI Engineer OS -- UI/UX Design Specification for Antigravity

## Mission

Build the world's most engaging GenAI & Agentic AI learning platform.

The platform must **not feel like documentation** or a traditional LMS.
It should feel like a premium product inspired by Apple, Cursor,
Brilliant.org, Figma, Linear and Duolingo.

Primary goals:

-   Make users curious to open the next lesson.
-   Maximize long-term retention.
-   Minimize cognitive overload.
-   Make learning feel like exploration.

------------------------------------------------------------------------

# Core Design Principles

1.  Every page should answer **"What should I learn next?"**
2.  Every interaction should reward the learner.
3.  Motion should guide attention, not distract.
4.  Visuals should teach concepts.
5.  Every topic should feel like unlocking a new ability.

------------------------------------------------------------------------

# Motion Design

Every interaction should have motion.

No static interface.

## Page Load

-   Fade in
-   Cards slide upward
-   Progress bars animate
-   Hero illustration slowly appears

Duration: 400--600ms

------------------------------------------------------------------------

Cards

-   Lift 4px
-   Scale 1.02
-   Border glow
-   Shadow increase

-   Watch progress

-   Deep dive (20--40 min)

-   Enterprise talk
-   Research walkthrough
