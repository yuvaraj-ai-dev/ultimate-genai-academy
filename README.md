## 🚀 How to Launch the Learning Portal

We have built a custom, interactive Web Application Dashboard inside the `learning-portal/` directory to help you visualize your 90-day progress, read visual analogies, access recommended video guides, complete checklists, and prepare for interviews.

To launch the dashboard locally:

1. **Open your terminal** and navigate to the portal directory:
   ```bash
   cd learning-portal
   ```

2. **Start the Next.js development server**:
   ```bash
   npm run dev
   ```

3. **Open the browser** and navigate to:
   [http://localhost:3000](http://localhost:3000)

Your learning progress (completed days and project checklists) is saved automatically in your browser's **localStorage** across sessions.
